📌 FocusTrack — Professional To-Do App

A modern, responsive To-Do app with categories, filters, persistent storage, and personalization.  
Built with HTML, CSS, JavaScript— no frameworks required. 

✨ Features
- 🗂️ Categories default(Inbox, Work, Personal, Custom)
- ✅ Task Filters(All, Pending, Completed)
- 💾 LocalStorage Persistence(your tasks stay even after reload)
- 🖋️ Edit Name(personalized greeting)
- 📱 Responsive Layout
- 🎨 Clean UI with Dark Mode

🚀 Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (DOM, LocalStorage)
